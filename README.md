# Weather Dashboard
  
  ## Description 
  Gives current and five day forecast for requested location. Stores prior searches in local storage.
  ## Table of Contents
  * [Usage](#usage)
  * [Questions](#questions)
  
  ## Usage 
  Enter the city you would like to find weather information for and hit the search button.
